# NovaAI Landing Page

A modern one-page landing page for an AI productivity platform built with HTML, CSS, and JavaScript.

## Features

- Responsive hero section
- Feature cards
- About and stats sections
- Pricing section
- FAQ section
- Contact form
- Smooth scrolling and subtle animations

## Project Structure

- index.html — main page structure
- css/style.css — main stylesheet
- css/responsive.css — responsive styles
- js/script.js — page interactions
- images/ — project images and illustrations

## Run Locally

Open the project folder and start a simple local server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Notes

- The page uses a local SVG image so it works reliably in local preview.
- If you are using VS Code, Live Server can also be used to preview the page.
