# Portfolio

This portfolio website shows the following sections:

- Home
- About
- Education
- Skills
- Projects
- Resume
- Contact

## Project screenshots

The project cards in this portfolio are displayed using styled placeholders rather than screenshot images.

If you want to add screenshots later, put them in `Portfolio/images/` and update `Portfolio/index.html` accordingly.

## Resume

Place a real PDF named `resume.pdf` in `Portfolio/` so the Download Resume button works.
